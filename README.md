# NodeJS API

### This is a very basic Music API builted with Nodejs with the following features which you can use it as a boilerplate 


## Features
- [x] CRUD Operations (Create, Read, Update, Delete)
- [x] Database
- [x] Middleware
- [ ] User Management
- [ ] Security
